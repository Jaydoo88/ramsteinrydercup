What needs to change

1. Keep the player cards aligned to the top

The top of the player cards should stay aligned with the top of the left panel.

2. Reduce the vertical expansion of the left panel

Tighten spacing between paragraphs.
Reduce excessive padding at the top and bottom.
Shorten the divider spacing.
Make the “Role” section sit closer to the content above it.

3. Limit how much text can visually grow

Use a more compact layout for the left panel copy.
Reduce line-height slightly so the text block is tighter but still readable.
Consider clamping some of the descriptive text to a set number of lines on desktop if needed.
The left panel should feel like a summary card, not a long article.

4. Make the whole section height feel intentional

The left panel should visually match the height rhythm of the player cards.
Avoid large blank space at the bottom of the left panel.
If needed, make the left panel a fixed/min-height card with content vertically distributed more evenly.

5. Improve the layout structure
Use a layout where:

the left panel is one column
the player cards sit in the adjacent columns
all columns align from the top
cards do not get pushed down by taller neighboring content

A CSS grid layout with align-items: start is probably better here than anything that is stretching based on content height.
